# 📊 ConnectaTel - Análisis Exploratorio de Clientes

## 📌 Descripción del proyecto

Este proyecto presenta un análisis exploratorio de datos (EDA) realizado sobre la base de clientes de ConnectaTel. El objetivo fue comprender el comportamiento de los usuarios, identificar segmentos de clientes, analizar patrones de consumo y generar recomendaciones de negocio basadas en datos.

Este proyecto fue desarrollado como parte del programa de Ciencia de Datos de TripleTen, aplicando técnicas de limpieza de datos, análisis exploratorio y visualización utilizando Python.

---

## 🎯 Objetivos

- Evaluar la calidad del conjunto de datos.
- Analizar la distribución demográfica de los clientes.
- Identificar segmentos según el nivel de consumo.
- Detectar valores atípicos (Outliers).
- Extraer hallazgos que apoyen la toma de decisiones del negocio.

---

## 📂 Conjunto de datos

El proyecto utiliza tres conjuntos de datos:

- **users_latam.csv**: información demográfica de los clientes.
- **plans.csv**: características de los planes ofrecidos.
- **usage.csv**: registros de consumo de llamadas, mensajes y minutos.

---

## 🔎 Proceso de análisis

Durante el proyecto se realizaron las siguientes etapas:

- Limpieza y preparación de datos.
- Tratamiento de valores faltantes.
- Corrección de inconsistencias.
- Análisis exploratorio de datos (EDA).
- Segmentación de clientes.
- Identificación de patrones de consumo.
- Detección de valores atípicos.
- Elaboración de conclusiones y recomendaciones para el negocio.

---

## 📈 Principales hallazgos

- La mayoría de los clientes pertenece al segmento de consumo medio.
- Los clientes de alto consumo representan una proporción pequeña, pero generan un volumen considerablemente mayor de uso.
- La edad no mostró una influencia significativa en la elección del plan contratado.
- Los valores atípicos correspondieron a comportamientos reales de ciertos clientes y no a errores de captura.

---

## 💡 Recomendaciones

- Diseñar campañas de upselling para clientes de consumo medio.
- Implementar programas de fidelización para clientes de alto consumo.
- Evaluar nuevos planes dirigidos a usuarios con consumo intermedio.
- Fortalecer los procesos de validación y calidad de los datos.
